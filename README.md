## Dev development
### Step 1: Clone the project
### Step 2: Move to the project folder 
### Step 3: Set WEATHERSTACK_API_KEY
### Step 4: Run
```bash
docker-compose up -d
```

### Step 5: Visit
[https://localhost](https://localhost)
